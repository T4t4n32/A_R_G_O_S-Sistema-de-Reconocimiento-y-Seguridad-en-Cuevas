# Guía de uso de logos (ARGOS)

## Dónde se usan
- README del repo (banner).
- docs/ (portadas y anexos).
- Presentación del proyecto (FLL).
- Material para jueces (dossier con QR).

## Colores / fondos
- Fondo claro → preferir LOGO_ARGOS.png
- Fondo oscuro o fotografía → preferir SIN FONDO.png o SINFONDO_12.png

## Exportación (cuando diseñen)
- Mantener PNG para web.
- Para impresión: exportar también PDF/SVG si el software lo permite.
