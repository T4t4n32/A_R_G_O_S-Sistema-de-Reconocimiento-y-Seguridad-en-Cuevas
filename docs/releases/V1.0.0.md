# Release v1.0.0 — Baseline del proyecto

Fecha: 2026-03-10

## Resumen
Este release establece la base estable de ARGOS V1:
repositorio ordenado, documentación formal, y estructura de software orientada a Raspberry Pi 5.

## Qué incluye
- Estructura estable del repo (docs/software/hardware/assets/deploy/tests/datasets).
- Documentación por capítulos (1–5) y plantillas operativas.
- Manual del equipo para aprender y explicar ARGOS con claridad.
- Archivos de comunidad: CONTRIBUTING, SECURITY, CODE_OF_CONDUCT.
- Política de versionado y changelog.

## Estado del prototipo
- Prototipo en desarrollo: sensores/LoRa/visión en progreso.
- Este release no declara “producto final”; declara “base estable” para construir sin reestructurar.
