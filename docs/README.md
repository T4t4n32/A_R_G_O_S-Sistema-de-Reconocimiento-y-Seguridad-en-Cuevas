# docs/ — Documentación ARGOS V1

Esta carpeta contiene la documentación base del proyecto **ARGOS V1** (CALIBOTS KAIROS).

## Estructura
- `tesis/` — documento tipo “proyecto” por capítulos (1–5).
- `arquitectura/` — diagramas y descripciones técnicas (bloques y flujos).
- `seguridad/` — umbrales, protocolo operativo y advertencias.
- `plantillas/` — formatos para cronograma, BOM, plan de pruebas, bitácora, entrevistas.
- `referencias/` — bibliografía (APA/IEEE).
- `identidad/` — guía de uso de activos visuales.

## Nota importante
ARGOS V1 es un prototipo educativo. Los umbrales y normas se usan como **referencia de ingeniería**; no reemplaza instrumentación certificada ni autoriza entrada real a cuevas.
