# Advertencias de seguridad (obligatorias) — ARGOS V1

1) ARGOS V1 es un prototipo educativo y **no certifica** la seguridad de una cueva.  
2) No usar MQ-135 como único criterio de “entrar/no entrar” (no es selectivo ni mide oxígeno).  
3) Los umbrales OSHA/NIOSH se usan como referencia; las decisiones reales requieren instrumentos certificados y supervisión responsable.
4) Pruebas físicas deben hacerse en entornos controlados y con adultos responsables.

Referencias:
- OSHA/29 CFR 1910.146: https://www.law.cornell.edu/cfr/text/29/1910.146
- NIOSH 86-110: https://www.cdc.gov/niosh/docs/86-110/default.html
