# Plantilla — Guía de entrevista (10 preguntas)

**Objetivo:** validar problema, utilidad y requisitos de seguridad.

1) ¿Qué riesgos considera más críticos en una cueva/espacio confinado?
2) ¿Qué mediciones mínimas haría antes de entrar?
3) ¿Qué señales o condiciones le harían detener la exploración?
4) ¿Qué tan útil sería recibir telemetría (estado/alertas) desde el interior?
5) ¿Qué tan común es la pérdida de comunicación en entornos similares?
6) ¿Qué información necesita un rescatista/equipo externo para apoyar una operación?
7) ¿Qué elementos considera “hallazgos” relevantes que deberían registrarse?
8) ¿Qué errores le preocuparían en un sistema automático (falsos positivos/negativos)?
9) ¿Qué requisitos de robustez física esperaría (agua/polvo/golpes)?
10) Si pudiera mejorar un aspecto del proceso actual, ¿cuál sería?
