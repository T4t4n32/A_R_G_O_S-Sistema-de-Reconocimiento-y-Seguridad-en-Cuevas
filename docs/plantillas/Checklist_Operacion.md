# Checklist — Operación ARGOS V1 (pre/durante/post)

## Pre-entrada (ANTES)
- [ ] Batería cargada / fuente estable
- [ ] Sensores detectados (I²C/SPI ok)
- [ ] LoRa enlaza (mensaje de prueba recibido)
- [ ] Cámara funciona (preview / captura)
- [ ] Iluminación activa
- [ ] Log de inicio creado

## Durante (OPERACIÓN)
- [ ] Monitoreo continuo (lecturas actualizan)
- [ ] Alertas se registran (log)
- [ ] Telemetría sale por LoRa
- [ ] Capturas se guardan ante “anomalía”
- [ ] Si alerta ROJA → detener/retornar (simulación)

## Post (DESPUÉS)
- [ ] Reporte final generado
- [ ] Logs exportados/respaldados
- [ ] Evidencias (fotos/videos) organizadas
- [ ] Bitácora de iteraciones actualizada
