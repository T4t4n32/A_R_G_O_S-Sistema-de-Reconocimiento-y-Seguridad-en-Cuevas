# Safety & Security Notice (ARGOS)

ARGOS V1 is an educational prototype. It does **not** certify safety for real cave entry.

## Safety principles
- Test only in controlled environments.
- Do not rely on MQ-135 as a life-safety oxygen measurement.
- Use qualified supervision for any field-like tests.

## Reporting
Report any safety concerns to the coaches/mentors and log them in `docs/seguridad/`.
