# Code of Conduct — CALIBOTS KAIROS (ARGOS)

## Principios
- Respeto, colaboración y comunicación profesional.
- Seguridad por encima de la demostración.
- Integridad: datos reales, sin inventar métricas ni evidencias.

## No permitido
- Acoso, humillación, discriminación.
- Compartir información privada sin autorización.
- Presionar a alguien a realizar pruebas inseguras.

## Reporte
Incidentes se reportan a coaches/mentores.
